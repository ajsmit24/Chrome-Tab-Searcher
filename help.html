Welcome to the help page!

1. The help button

When clicked it brings you to this help page. A similar button with a < is in the top left of the this page to take you back to the man page.

2. The search bar

Enter a string and hit the "enter" key on your keyboard or the "search" button(6). Then a list of resualts will be displayed by clicking an item on this list you can navigate to that tab.

3. Where to search

By leaving the URL box checked any tab containing a match in its URL will be included in the list of resualts.

By leaving the Title box checked any tab containing a match in its title will be included in the list of resualts.

4. The type of match

If Exact is selected only tabs with a title/URL that is the exact same as the search input will be included in the list of resualts.

If contains is selcted an tab with a title/URL that conatins the search input will be included in the list of resualts.

5. Case Matters

If selected case will be considered in determining if a title/URL contains or exactly mmatches the search input.

6. The Search Button

When clicked the search is exicuted.
